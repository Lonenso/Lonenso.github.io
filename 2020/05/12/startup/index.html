记Hexo + Github Pages 建博客

今天心血来潮, 想要重新用github建立个博客. 官方的教程不错. 这里记录下遇到的问题

通过两仓库或两分支的操作, 把博客内容区和工作区分开, 同时跟踪和保留工作现场(备份).

这里我选择两仓库, 工作区私有, 内容区公开. 使用官方的工具, 非常方便.

搜索引擎收录问题, 谷歌搜索

这时候网站没有被收录, 搜索结果中有个 google search console, 根据提示完成网站认证的添加.

我选择的是将html文件放到站点下, 使得https://lonenso.github.io/xxxx.html 能够访问到该文件.

也就是说, 本地https://localhost:4000/xxxx.html 能够访问到该原文件, 不是被渲染后的, 配置skip_render.
